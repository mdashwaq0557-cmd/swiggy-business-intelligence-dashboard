# swiggy-business-intelligence-dashboard
A comprehensive Power BI Business Intelligence Dashboard analyzing customer behavior, restaurant
**Project Overview:**
This project presents a comprehensive Business Intelligence solution developed using Power BI to analyze operational and business performance across multiple dimensions of a food delivery platform.

**The dashboard provides insights into:**
>Customer Intelligence
>Restaurant Performance
>Delivery Operations
>Revenue Analytics
>Geographic & Market Analysis
>Executive Overview

**1. Executive Overview**
